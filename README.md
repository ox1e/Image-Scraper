# Тестовое задание для ТЕХНЕЗИС

## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/ox1e/Image-Scraper.git
2. Перейдите в директорию проекта:

   ```bash
   cd Image-Scraper

3. Установите зависимости:
   
   ```bash
   composer install
4. Запустите сервер:

   ```bash
    symfony server:start
